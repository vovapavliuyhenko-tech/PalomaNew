PROJECT RULES — PALOMA FLOWERS

1. Структурный референс — plombirflowers.ru. Логика блоков, UX, e-commerce flow.
2. Визуальный стиль — Paloma. Не копировать Plombir визуально.
3. Стек: Next.js (App Router) + TypeScript + Tailwind + Framer Motion + GSAP ScrollTrigger + Zustand.
4. Цвета: #FFFDF8 (фон), #282828 (темные секции), #7F0D0E (бордовый), #F0522B (оранжевый), #E296B2, #FEECDE.
5. Логотип всегда текстовый: PALOMA / flowers coffee you.
6. Контейнер: max-width 1440px, padding 40px (60px на large), 16px mobile.
7. Вертикальные отступы: 120px большие секции, 80px средние, 56px mobile.
8. Сетка каталога: 4/2/1 колонки, gap 24/20/16.
9. Фото товара: aspect-ratio 3/4, object-fit cover.
10. Запрещено: мусорные placeholder, дешевые градиенты, хаотичные отступы, копия Plombir визуально, упрощение e-commerce логики.
11. Все товары и категории — в data-файлах с типизацией.
12. Работать только в рамках текущего этапа. Не трогать другие этапы без явного указания.

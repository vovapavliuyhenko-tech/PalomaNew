## Продолжение (после согласования пользователя)

**Сетка:** массовая замена связки `max-w-[var(--container-max)]` + `px-[var(--container-pad)]` на **`container mx-auto`** по дереву `src` (tsx); точечные правки без скрипта: `PageHero` — обёртка `container` + внутренний блок с опциональным `maxWidthClass`; `HomeCafeSection`; mobile-ветка `FulfillmentProcess`; `QuickCategories` — горизонтальный скролл с `-mx`/`px`, синхронизированными с `--container-padding-{mobile,tablet,desktop}`.

**Анимации / премиум-UX:** `ScrollReveal` добавлен на страницы **блога** и **контактов**; на **мероприятия** — для блоков «услуги» и «портфолио». Блок квиза переведён на фон `--color-oil` и структуру `container` → `max-w-3xl`.

**Контент events:** убраны эмодзи в карточках услуг (замена на редакционную нумерацию 01–06 и типографику display/caption).

**Проверка:** `npm run build` — успешно.

**Дополнительно:** `ScrollReveal` + обёртки `container mx-auto max-w-*` с вертикальными `--space-xl` / `--space-xxl` на страницах **доставка**, **FAQ**, **подписка** (два блока), **уход**, **оплата**, **о нас**, **подарки**, **согласие**, **политика**, **оферта**, **каталог** (`CatalogContent`), **корзина** (непустая), **checkout**, **пост блога** (`[slug]`), **свадебная копилка** (шаги + форма). На **уход за букетом** эмодзи в советах заменены на иконки Lucide (в духе блока «Оплата»). Клиент копилки: `container` вместо ручного `px-[var(--container-pad)]`.

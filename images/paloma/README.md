# Медиа PALOMA

Скопируйте финальные фото из папки **«дизайн»** в подпапки:

| Папка | Файлы | Где используется |
|-------|--------|------------------|
| `hero/` | `hero-main.jpg` или `.webp` | Главная, hero |
| `catalog/` | `c01.jpg` … по slug | Каталог, карточки |
| `coffee/` | `hero.jpg`, позиции меню | Кофейня |
| `wedding/` | `hero.jpg`, портфолио | weddings.html |
| `subscription/` | `bg.jpg` | Блок подписки |
| `reviews/` | `review-1.jpg` … | Отзывы |

Пути настраиваются в `paloma-media-paths.js`.  
Если файла нет — остаётся градиент-placeholder (сайт не ломается).
